# Usage of Sira UI

Sira UI 被设计为基于 TailwindCSS 的 UI 库，适用于任何类型的应用程序，这意味着您可以使用类似 Bootstrap 的 Sira UI 或其他类似的 UI 库。

对于任何人来说，使用组件都非常简单，只需使用 `class` 属性将类添加到 HTML 元素即可。

这里我们展示了两个简单的例子。

- 按钮样例

```html
<button class="btn solid success">Hello World</button>
```

<button className="btn solid success">Hello World</button>

- 输入框样例

```html
<input class="input danger" placeholder="Hello World" />
```

<input className="!input !ring-offset-transparent danger" placeholder="Hello World" />
