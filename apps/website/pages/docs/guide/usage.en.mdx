# Usage of Sira

Sira UI is designed as a TailwindCSS based UI library for any type application,
so that means you can use Sira UI like Bootstrap or another similar UI library, but with the power of TailwindCSS.

For use the components it's really easy for anyone, just use the `class` prop to add the classes to the HTML element.

Here we show up 2 simple examples.

- Button Example

```html copy
<button class="btn solid success">Hello World</button>
```

<button className="btn solid success">Hello World</button>

- Input Example

```html copy
<input class="input danger" placeholder="Hello World" />
```

<input className="!input !ring-offset-transparent danger" placeholder="Hello World" />
