# Checkbox

## Default

```html
<div class="flex gap-4">
  <input checked class="bw checkbox" type="checkbox" />
  <input class="success checkbox" type="checkbox" />
  <input checked class="warn checkbox" type="checkbox" />
  <input class="danger checkbox" type="checkbox" />
</div>
```
