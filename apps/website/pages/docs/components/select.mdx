# Select

## Default

```html
<select class="select danger">
  <option>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
</select>
```

<select className="select danger">
  <option>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
</select>
